# 基于Reed-Solomon码的二维码纠错能力研究与可视化验证系统

## 项目简介

本项目是一个基于Reed-Solomon码的二维码纠错能力研究与可视化验证系统，旨在通过实验和可视化方式展示不同纠错等级的二维码在面对遮挡或污染时的鲁棒性。

## 功能特点

- 生成不同纠错等级（L/M/Q/H）的二维码
- 支持不同版本（1-10）的二维码生成
- 随机遮挡或指定区域遮挡二维码
- 实时解码并反馈结果
- 可视化遮挡像素分布
- 分析不同遮挡面积对解码成功率的影响
- 分析不同位置遮挡的敏感性

## 使用说明

1. **生成二维码**：
   - 在左侧输入要编码的内容
   - 选择纠错等级（L/M/Q/H）
   - 选择二维码版本（1-10）
   - 点击"生成二维码"按钮

2. **应用遮挡**：
   - 选择遮挡方式（随机遮挡）
   - 调整遮挡百分比（0-40%）
   - 点击"应用随机遮挡"按钮

3. **解码二维码**：
   - 点击"解码二维码"按钮
   - 查看解码结果（成功/失败）

4. **分析功能**：
   - 点击"分析遮挡面积影响"查看不同遮挡面积的成功率曲线
   - 点击"分析位置敏感性"查看不同位置遮挡的影响

## 技术原理

### Reed-Solomon码

Reed-Solomon码是一种强大的纠删码，具有以下特点：

- 定义在有限域GF(256)上
- 块码结构，包含数据符号和冗余符号
- 错误检测与纠错能力公式：t = floor((n - k) / 2)
  - t：可纠正的错误符号数
  - n：码字长度
  - k：数据符号长度

### QR码的纠错机制

QR码使用Reed-Solomon码进行纠错，具有四种纠错等级：

- **L级**：约可纠正7%的数据错误
- **M级**：约可纠正15%的数据错误
- **Q级**：约可纠正25%的数据错误
- **H级**：约可纠正30%的数据错误

### 解码过程

1. 图像定位
2. 采样与网格化
3. 解RS码获取原始数据
4. 验证解码结果

## 实验设计

### 实验一：不同遮挡面积的影响

- 生成同一个二维码
- 分别施加0%-40%遮挡
- 测试不同纠错等级的成功率

### 实验二：遮挡位置敏感性

- 遮挡定位图案
- 遮挡数据区
- 遮挡校准图案
- 对比成功率与失败原因

### 实验三：不同版本二维码的鲁棒性对比

- 版本1 vs 版本5 vs 版本10
- 数据块数量、整体冗余差异
- 成功率变化分析

## 结果分析

通过本系统，您可以观察到以下现象：

1. **纠错等级影响**：
   - 高纠错等级（Q/H）在高遮挡率下仍能保持较高的解码成功率
   - 低纠错等级（L/M）在遮挡率超过一定阈值后解码成功率急剧下降

2. **位置敏感性**：
   - 定位图案的遮挡对解码影响最大
   - 数据区域的遮挡可以通过RS码进行纠正
   - 定时图案的遮挡会影响二维码的识别

3. **版本影响**：
   - 高版本二维码包含更多的数据块和冗余信息
   - 局部损伤对高版本二维码的影响相对较小
